What is Force Calendar Core?

The Technical Foundation

Force Calendar Core is the computational engine that powers calendar operations. Think of it as the mathematical brain that handles time calculations, event storage, and calendar logic - without any visual components.

While traditional calendar libraries bundle everything together, Force Calendar Core focuses solely on the algorithmic challenges: managing time across zones, detecting conflicts, parsing recurrence rules, and optimizing event queries. This separation allows you to build exactly the interface you need, whether that's a React component, a Salesforce Lightning app, or a command-line tool.

How It Works

Under the Hood

The engine uses spatial data structures to index events by time, enabling constant-time lookups regardless of dataset size. Events are stored in a multi-dimensional index that maps time ranges to memory locations, similar to how databases index records for fast retrieval. 

// Initialize the calendar engine
import { Calendar } from '@forcecalendar/core';

const calendar = new Calendar({
    timeZone: 'America/New_York',
    weekStartsOn: 0, // Sunday
    businessHours: { start: '09:00', end: '17:00' }
});

// Add an event with recurrence
calendar.addEvent({
    id: 'standup-1',
    title: 'Daily Standup',
    start: new Date('2025-01-15T09:00:00'),
    end: new Date('2025-01-15T09:15:00'),
    recurrence: 'FREQ=DAILY;BYDAY=MO,TU,WE,TH,FR'
});

// Query events with O(1) performance
const todayEvents = calendar.getEventsForDay(new Date());
const conflicts = calendar.detectConflicts(newEvent);

Core Capabilities

What Sets It Apart

O(1)

Spatial Indexing

Events are indexed using R-tree spatial data structures, providing constant-time lookups regardless of calendar size. Query millions of events as fast as querying ten.

TZ

Timezone Intelligence

Full IANA timezone database with DST handling. Automatically converts between zones, handles daylight saving transitions, and manages historical timezone changes.

RFC

Standards Compliance

Implements RFC 5545 (iCalendar) specification completely. Parse and generate RRULE recurrence patterns compatible with Google Calendar, Outlook, and Apple Calendar.

SF

Salesforce Optimized

Built specifically for Locker Service compatibility. No DOM dependencies, no global pollution, fully sandboxed execution. Works seamlessly in Lightning Web Components.

MEM

Adaptive Memory

Intelligent memory management with LRU caching and automatic garbage collection. Scales from embedded devices to enterprise servers without configuration.

ICS

Universal Exchange

Import and export ICS/iCalendar files. Exchange data with any calendar system. Preserves all metadata including custom properties and extended attributes.
